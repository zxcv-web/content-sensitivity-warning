== Description ==
This plugin provides a non-javascript-dependent popup for Wordpress installs that may have culturally sensitive content.

Many Australian websites may contain images of indigenous people. For some cultural groups it may be insensitive to show images and sounds of people that are deceased.

For more information on cultural protocols relating to deaths in Indigenous communities https://apps.indigenous.gov.au/cultural_protocol.htm

This lightweight plugin is aimed to be a simple way of meeting requirements for sensitive content. Javascript is not required for functionality but is used to enhance the user experience.

== Installation ==
Download from github here:

https://github.com/zxcv-web/content-sensitivity-warning/

Install and activate as per other Wordpress plugins.
